# 0x05. Python - Exceptions

An exception is an event, which occurs during the execution of a program, that disrupts the normal flow of the program's instructions[oracle.com](https://docs.oracle.com/javase/tutorial/essential/exceptions/definition.html). In this lesson, we explored errors and exceptions in Python.

## Errors
Errors or mistakes in a program are often referred to as bugs. They are almost always the fault of the programmer. Errors can be classified into three major groups:

- Syntax errors
- Runtime errors
- Logical errors

### Syntax Errors
Syntax errors are mistakes associated with the wrong usuage of the syntax of a language.

### Runtime Errors
There are erros that occur during the execution of a program. These kinds of errors cause the program execution to halt abruptly. Runtime errors therefore causes programs to crash.

### Logical Errors
These are errors that cause a program to produce wrong output. The program will execute without any syntax or runtime errors but will not produce the expected or correct output.

## Debugging
The process of finding and eliminating errors is called debugging.
